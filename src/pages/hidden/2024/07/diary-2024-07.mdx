猜猜我今天没读论文没科研没工作在干什么，在调色配 `delta`（；之前感觉它默认长得不好看，今天才想着来配一下（，结合了 GitHub 的 diff 和 gruvbox 配色。

主要是发现 `tiny-code-action.nvim` 可以用 delta，试了一下感觉不如 `actions-preview.nvim`，然后再一看，原来 `actions-preview.nvim` 也可以 delta。太典了，新发现的 plugin 作用是更新旧 plugin 的 config（

```ini
[delta]
    dark = true
    navigate = true
    side-by-side = true
    syntax-theme = gruvbox-dark

    file-added-label = 🟩🟩
    file-copied-label = 📋
    file-modified-label = 🟥🟩
    file-removed-label = 🟥🟥
    file-renamed-label = ➡️
    file-decoration-style = ul ol "#fff1d6"
    file-style = "#ffdd99"

    hunk-header-decoration-style = box "#484848"
    hunk-header-file-style = italic "#928374"
    hunk-header-style = file syntax italic

    line-numbers = true
    line-numbers-left-format = "{nm:^5}"
    line-numbers-right-format = "{np:^5}"
    line-numbers-zero-style = "#928374"
    line-numbers-plus-style = "#fbf1c7" "#2a4c30"
    line-numbers-minus-style = "#fbf1c7" "#663432"

    zero-style = syntax
    plus-style = syntax "#293a2c"
    minus-style = syntax "#472e2d"
    plus-emph-style = "#fbf1c7" "#2b6436"
    minus-emph-style = "#fbf1c7" "#903c38"
    map-styles = bold purple => syntax "#4c2a4c", bold cyan => syntax "#34345a"
```
